# Plan 01-01: WhatsApp Design Token

## Objective

Add `--whatsapp` and `--whatsapp-dark` tokens to `:root` in style.css and replace every hardcoded `#25D366` and `#20BA5A` occurrence with the token references. This is the lowest-risk change in the phase — pure search-and-replace with zero visual change and zero cascade impact. It must run first so that the `@layer` plan (Wave 2) can reference token names rather than hex values in its verification grep.
