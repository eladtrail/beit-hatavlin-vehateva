# Plan 01-02: GPU Animations + Reduced-Motion Guard

## Objective

Two tightly related fixes that must land together: (1) convert the cart sidebar animation from layout-triggering `right` property to GPU-composited `transform: translateX()` (FOUND-03), and (2) fix the `scroll-bounce` keyframe to use `transform` instead of `top` — then add `@media (prefers-reduced-motion: reduce)` blocks to both CSS files to suppress all looping animations and instant-reveal transition elements (FOUND-04). Both tasks touch the same mental model (animation properties) and their verification overlaps, so they ship in one wave-1 plan.

---

## Tasks

<task id="1-02-01" type="edit">
<description>Convert #cartSidebar slide animation from right:-100%/right:0 (layout) to transform:translateX(100%)/translateX(0) (GPU composited). The right:0 static position is retained; only the animation target changes.</description>
<file>css/shop.css</file>
<changes>
Find (lines 442-458):
#cartSidebar {
  position: fixed;
  top: 0;
  right: -100%;
  width: 100vw;                       /* full-width on mobile */
  height: 100dvh;
  background: var(--white);
  z-index: 400;
  display: flex;
  flex-direction: column;
  transition: right .35s cubic-bezier(.4, 0, .2, 1);
  box-shadow: var(--shadow-xl);
}
#cartSidebar.cart-open { right: 0; }

/* On larger phones / tablet: panel not full width */
@media (min-width: 480px) { #cartSidebar { width: min(420px, 96vw); } }

Replace with:
#cartSidebar {
  position: fixed;
  top: 0;
  right: 0;                           /* always at right edge — transform handles visibility */
  width: 100vw;                       /* full-width on mobile */
  height: 100dvh;
  background: var(--white);
  z-index: 400;
  display: flex;
  flex-direction: column;
  transform: translateX(100%);        /* hidden off-screen to physical right — GPU composited */
  transition: transform .35s cubic-bezier(.4, 0, .2, 1);
  box-shadow: var(--shadow-xl);
}
#cartSidebar.cart-open { transform: translateX(0); }

/* On larger phones / tablet: panel not full width */
@media (min-width: 480px) { #cartSidebar { width: min(420px, 96vw); } }
</changes>

  .reveal-left {
    opacity: 1;
    transform: none;
    transition: none;
  }
}
