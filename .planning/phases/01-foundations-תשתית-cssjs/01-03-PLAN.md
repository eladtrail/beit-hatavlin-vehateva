# Plan 01-03: LCP Preload, Font Weight Trim, CSV Timeout

## Objective

Three independent low-risk improvements that touch HTML and JS only. They are grouped in Wave 2 because they are safe to execute in any order after Wave 1 is verified stable, and batching them minimizes review cycles. FOUND-05 adds hero image preload links to `index.html <head>`. FOUND-06 removes unused font weight variants from both HTML files (saves 2 HTTP requests). FOUND-07 adds an AbortController-based 5-second timeout to the Google Sheets CSV fetch in shop.js.

<task id="1-03-04" type="edit">
<description>Add AbortController-based 5-second timeout to the loadFromSheet() function in shop.js. Replace the bare fetch() call (line 78) with a timeout-guarded fetch that aborts after 5000ms and falls back gracefully to the already-rendered demo products. The AbortController pattern actually cancels the network request rather than just ignoring a late response.</description>
<file>js/shop.js</file>
<changes>
Find (lines 75-90):
async function loadFromSheet() {
  if (!SHEET_CSV_URL) return;
  try {
    const resp = await fetch(SHEET_CSV_URL);
    if (!resp.ok) return;
    const text = await resp.text();
    if (text.trim().startsWith('<')) return; // קיבלנו HTML — לא CSV
    const parsed = parseCSV(text);
    const active = parsed.filter(p => p.active);
    if (!active.length) return; // אין מוצרים ב-Sheet — נשאר עם דמו
    allProducts = active;
    applyFilters(); // רענן תצוגה עם נתוני Sheet
  } catch (err) {
    console.warn('⚠️ Sheet load failed:', err.message, '— keeping demo products');
  }
}

Replace with:
async function loadFromSheet() {
  if (!SHEET_CSV_URL) return;
  const controller = new AbortController();
  const timeoutId = setTimeout(() => controller.abort(), 5000);
  try {
    const resp = await fetch(SHEET_CSV_URL, { signal: controller.signal });
    clearTimeout(timeoutId);
    if (!resp.ok) return;
    const text = await resp.text();
    if (text.trim().startsWith('<')) return; // קיבלנו HTML — לא CSV
    const parsed = parseCSV(text);
    const active = parsed.filter(p => p.active);
    if (!active.length) return; // אין מוצרים ב-Sheet — נשאר עם דמו
    allProducts = active;
    applyFilters(); // רענן תצוגה עם נתוני Sheet
  } catch (err) {
    clearTimeout(timeoutId);
    console.warn('⚠️ Sheet load failed:', err.message, '— keeping demo products');
  }
}
</changes>
